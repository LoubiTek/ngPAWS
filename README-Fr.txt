ngPAWS système auteur
=======================

ngPAWS (prononcé n-g-paws) est un système auteur basé sur le système Professional Adventure Writing System.
Le nom ngPAWS signifie "nouvelle génération PAWS".

PAWS était un système auteur largement utilisé au Royaume-Uni et en Espagne dans les années 80 et 90, étant la version la plus connue de Sinclair ZX Spectrum, bien qu'il existe des versions pour CP/M (Amstrad CPC, PCW) et IBM PC. En outre, il existe plusieurs clones de PAW
fait dans les années 90, 2000 et 2010.

Qu'est-ce qui différencie ngPAWS ?
----------------------------------
Ce qui rend ngPAWS différent, c'est qu'il ne génère pas de fichier exécutable, mais un site Web pouvant être exécuté dans n'importe quel navigateur moderne. Ainsi, les aventures textuelles (alias fiction interactive) réalisées avec ngPAWS peuvent être exécutées sur n'importe quel serveur Web ou localement.

Construires
-----------

Pour les versions précompilées pour Windows, Linux (x86 et arm) et OSX, consultez notre page de téléchargements:

http://www.ngpaws.com#Downloads

Veuillez noter que les paquets précompilés incluent également txtpaws par Baltasarq (https://github.com/Baltasarq/txtpaws), un outil très utile pour les programmeurs/auteurs de ngPAWS.
