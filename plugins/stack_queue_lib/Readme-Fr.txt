Nouveau condact:

queueno/stackno = toute valeur 1-4. Si vous avez besoin de plus d'empilements/files d'attentes, éditez le fichier stack_queue_lib.jsp et changez la valeur MAX_QUEUE ou MAX_STACK.

PUSH setno value => Ajouter une valeur à une pile
POP setno flagno => Affiche la valeur d'une pile et la stocke dans le drapeau (dernier entré, premier sorti)

QPUSH setno value => Ajouter une valeur à une file d'attente
QPOP setno flagno => Affiche la valeur d'une file d'attente et la stocke dans le drapeau (premier entré, premier sorti)
