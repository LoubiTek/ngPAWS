Cette bibliothèque implémente une méthode simple pour avoir de longues descriptions d'objets pouvant être affichés lors de l'examen.

Instructions:

1) Copiez le fichier .pgl dans le dossier jsl de votre installation ngPAWS.
2) Modifiez les textes de localisation pour qu'ils comportent deux textes séparés par un caractère dièse (#).
Par exemple, convertissez: "un coffre" en "un coffre # Un coffre lourd avec un lion sculpté dessus".
3) Assurez-vous que votre verbe "examiner" est le numéro 30 dans le vocabulaire. Sinon, changez "var exam_longdescription = 30;" dans longdescription_lib.jsp au bon numéro.
