Nouveau condact:

setno = toute valeur 1-4. Si vous avez besoin de plus d'ensembles, éditez le fichier sets_lib.jsp et changez la valeur de MAX_SETS.

SPUSH setno value => Ajouter une valeur à un ensemble
SPOP setno value => Supprime la valeur d'un ensemble
IN setno value => Condition réussite si la valeur est définie
NOTIN setno value => Condition réussite si la valeur n'est pas définie
